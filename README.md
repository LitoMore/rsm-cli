# rsm-cli

Resume CLI

## Install

```shell
npm i -g rsm-cli
```

### Installed globally

```
$ rsm --help

  Usage
    $ rsm <resume>

  Example
    $ rsm LitoMore
    $ rsm LitoMore/LitoMore
    $ rsm https://example.com/resume.json
    $ rsm path/to/resume.json
```

### With `npx`

```shell
npx rsm-cli <resume>
```
